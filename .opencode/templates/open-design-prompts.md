# Biblioteca de Prompts para Open Design

**Prompts prontos para usar no Open Design, organizados por tipo de projeto.**

---

## 🌐 Landing Pages

### Hero Sections

#### Hero Centralizado com Background
```
Crie uma hero section centralizada:
- Headline: "[INSERIR HEADLINE]"
- Subheadline: "[INSERIR SUBTÍTULO]"
- CTA: Botão "[TEXTO DO BOTÃO]"
- Background: [imagem / gradiente / cor sólida]
- Altura: 100vh (fullscreen) ou 600px
- Estilo: moderno, minimalista
- Cores: [paleta do projeto]
```

#### Hero Split (Texto Esquerda, Imagem Direita)
```
Crie uma hero section split:
- Coluna esquerda:
  - Headline: "[INSERIR HEADLINE]"
  - Subheadline: "[INSERIR SUBTÍTULO]"
  - CTA: Botão "[TEXTO DO BOTÃO]"
- Coluna direita:
  - Imagem: [hero image ou mockup]
- Proporção: 50/50 ou 60/40
- Alinhamento vertical: centralizado
```

#### Hero com Vídeo de Fundo
```
Crie uma hero section com vídeo:
- Headline: "[INSERIR HEADLINE]"
- Subheadline: "[INSERIR SUBTÍTULO]"
- CTA: Botão "[TEXTO DO BOTÃO]"
- Background: vídeo em loop (overlay escuro 50%)
- Altura: 100vh
- Texto: branco com sombra
```

### Seções de Benefícios

#### Grid 3 Colunas com Ícones
```
Crie uma seção de benefícios:
- Título da seção: "[TÍTULO]"
- Subtítulo: "[SUBTÍTULO OPCIONAL]"
- Grid: 3 colunas responsivo
- Para cada coluna:
  - Ícone: [ícone 1, 2, 3]
  - Título: [benefício 1, 2, 3]
  - Descrição: [descrição curta 1, 2, 3]
- Estilo: cards com hover effect
- Espaçamento: 40px entre cards
```

#### Grid 2 Colunas com Imagens
```
Crie uma seção de features:
- Título: "[TÍTULO]"
- Grid: 2 colunas
- Para cada feature:
  - Imagem: [screenshot ou ilustração]
  - Título: [feature 1, 2, 3, 4]
  - Descrição: [descrição 1, 2, 3, 4]
- Layout: alternado (imagem esquerda/direita)
```

### Pricing

#### Tabela de Preços 3 Planos
```
Crie uma seção de preços:
- Título: "Escolha seu plano"
- Subtítulo: "Comece grátis, escale quando precisar"
- Grid: 3 colunas
- Para cada plano:
  - Nome: [Básico / Pro / Enterprise]
  - Preço: [R$ X/mês]
  - Features: [lista de 5-7 features]
  - CTA: Botão "[Começar / Upgrade / Contatar]"
- Destaque: plano Pro com badge "Mais Popular"
- Estilo: cards com sombra, Pro elevado
```

### FAQ

#### Accordion de Perguntas
```
Crie uma seção de FAQ:
- Título: "Perguntas Frequentes"

- Layout: accordion (expansível)
- Para cada pergunta:
  - Pergunta: [pergunta 1, 2, 3, 4, 5]

- Estilo: clean, com ícone + para expandir
```

## 📊 Apresentações

```
Crie um slide de capa:

- Subtítulo: "[SUBTÍTULO]"
- Data: [DD/MM/AAAA]

- Layout: centralizado

#### Capa com Imagem de Fundo
```
Crie um slide de capa com imagem:

- Subtítulo: "[SUBTÍTULO]"
- Background: [imagem relevante com overlay escuro 60%]

```

### Slides de Conteúdo

#### Slide de Problema
```
Crie um slide de problema:
- Título: "O Problema"
- Conteúdo:
  - Bullet 1: "[problema 1]"
  - Bullet 2: "[problema 2]"
  - Bullet 3: "[problema 3]"
- Dado impactante: "[estatística]" (destaque visual)
- Visual: ícone de warning ou imagem representativa
- Layout: split (texto 60%, visual 40%)
```

#### Slide de Solução
```
Crie um slide de solução:
- Título: "Nossa Solução"
- Conteúdo:
  - Benefício 1: "[benefício 1]" + ícone
  - Benefício 2: "[benefício 2]" + ícone
  - Benefício 3: "[benefício 3]" + ícone
- Visual: screenshot ou mockup do produto
- Layout: split (visual 60%, texto 40%)
```

#### Slide de Dados
```
Crie um slide de dados:
- Título: "[TÍTULO DO GRÁFICO]"
- Tipo de gráfico: [barras / linhas / pizza / área]
- Dados:
  - [Categoria 1]: [valor]
  - [Categoria 2]: [valor]
  - [Categoria 3]: [valor]
- Insight: "[1 linha com takeaway principal]"
- Fonte: [fonte dos dados, texto pequeno]
```

#### Slide de Depoimento
```
Crie um slide de depoimento:

- Nome: [Nome]

## 📈 Dashboards

### KPI Cards

#### Linha de 4 KPIs
```
Crie uma linha de KPI cards:
- KPI 1:
  - Nome: "[Nome do KPI]"
  - Valor: [valor atual]
  - Variação: [+X% ou -X%]
  - Ícone: [ícone relevante]
- KPI 2: [mesma estrutura]
- KPI 3: [mesma estrutura]
- KPI 4: [mesma estrutura]
- Layout: 4 cards em linha
- Estilo: cards com sombra, variação colorida (verde/vermelho)
```

### Gráficos

#### Gráfico de Linha Temporal
```
Crie um gráfico de linha:
- Título: "[Métrica ao longo do tempo]"
- Período: [últimos 30 dias / 6 meses / 1 ano]
- Dados: [inserir dados ou descrever tendência]
- Estilo:
  - Linha suave com área preenchida
  - Cor: [cor da marca]
  - Tooltip: mostrar valores ao hover
  - Legenda: [se múltiplas linhas]
- Eixos: X (tempo), Y (valores)
```

#### Gráfico de Barras Comparativo
```
Crie um gráfico de barras:
- Título: "[Comparação entre categorias]"
- Categorias: [cat1, cat2, cat3, cat4]
- Valores: [val1, val2, val3, val4]
- Estilo:
  - Barras verticais
  - Cor: [cor da marca com variações]
  - Labels: valores no topo das barras
- Eixos: X (categorias), Y (valores)
```

### Tabelas

#### Tabela de Dados com Filtros
```
Crie uma tabela de dados:
- Título: "[Título da tabela]"
- Colunas:
  - [Coluna 1]: [tipo: texto/número/data]
  - [Coluna 2]: [tipo]
  - [Coluna 3]: [tipo]
  - [Coluna 4]: [tipo]
- Dados: [5-10 linhas de exemplo]
- Funcionalidades:
  - Ordenação: clicável nos headers
  - Filtro: dropdown por coluna
  - Paginação: 10 itens por página
- Estilo: linhas alternadas, hover effect
```

## 📧 Email Templates

### Headers

#### Header Simples com Logo
```
Crie um header de email:
- Logo: [inserir logo, max-width 200px]
- Layout: centralizado
- Padding: 40px top/bottom
- Fundo: branco
- Borda inferior: 1px solid #eee
```

#### Header com Background Colorido
```
Crie um header de email:
- Logo: [inserir logo, branco]
- Título: "[TÍTULO DO EMAIL]"
- Layout: centralizado
- Padding: 60px top/bottom
- Fundo: [cor da marca]
- Texto: branco
```

### Corpos

#### Email de Texto Simples
```
Crie um corpo de email:
- Saudação: "Olá [Nome],"
- Parágrafo 1: "[INTRODUÇÃO + CONTEXTO]"
- Parágrafo 2: "[CONTEÚDO PRINCIPAL]"
- Parágrafo 3: "[BENEFÍCIO + CTA]"
- CTA: Botão "[TEXTO DO BOTÃO]"
- Assinatura:
  - "Abraços,"
  - "[Nome]"
  - "[Cargo]"
  - "[Empresa]"
- Layout: coluna única, max-width 600px
- Fonte: sans-serif, 16px, line-height 1.6
```

#### Email com Imagem
```
Crie um corpo de email com imagem:
- Saudação: "Olá [Nome],"
- Imagem: [imagem 600x300, centralizada]
- Título: "[TÍTULO]"
- Parágrafo: "[CONTEÚDO]"
- CTA: Botão "[TEXTO DO BOTÃO]"
- Layout: coluna única, max-width 600px
```

### Footers

#### Footer Simples
```
Crie um footer de email:
- Links sociais:
  - [LinkedIn] [Twitter] [Instagram] [Facebook]
- Link de descadastro: "Cancelar inscrição"
- Endereço: "[Endereço da empresa]"
- Layout: centralizado
- Fonte: 12px, cor cinza
- Padding: 40px top/bottom
- Fundo: #f5f5f5
```

## 🎨 Dicas Gerais

### Cores
```
Paleta de cores:
- Primária: [#HEX]
- Secundária: [#HEX]
- Accent: [#HEX]
- Texto: [#HEX]
- Fundo: [#HEX]
- Cinza claro: #f5f5f5
- Cinza médio: #999
- Cinza escuro: #333
```

### Tipografia
```
Fontes:
- Títulos: [Fonte Bold], 36-60pt
- Subtítulos: [Fonte Regular], 24-30pt
- Corpo: [Fonte Regular], 16-18pt
- Legenda: [Fonte Regular], 12-14pt
- Line-height: 1.6 para corpo, 1.2 para títulos
```

### Espaçamento
```
Padding:
- Seções: 80-120px top/bottom
- Cards: 40px
- Texto: 20px entre parágrafos
- Grid: 40px entre colunas

Max-width:
- Container: 1200px
- Texto: 800px
- Email: 600px
```
