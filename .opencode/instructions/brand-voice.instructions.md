# Brand Voice — Guia de Identidade Verbal

> Consistência de voz constrói reconhecimento de marca.
> Cada peça de conteúdo deve soar como se viesse da mesma pessoa.

---

## 1. Definindo a Voz da Marca

### Os 4 Pilares da Voz

Para cada marca/projeto, defina estes 4 pilares antes de produzir conteúdo:

| Pilar | Pergunta | Exemplo |
|-------|----------|---------|
| **Personalidade** | Se a marca fosse uma pessoa, como seria? | Confiante, acessível, inovadora |
| **Tom** | Como a marca fala? | Direto, empático, com humor sutil |
| **Valores** | O que a marca defende? | Transparência, excelência, impacto |
| **Propósito** | Por que a marca existe? | Transformar a forma como empresas se comunicam |

### Espectro de Voz

Posicione a marca em cada espectro (1-5):

```
Formal  ●○○○○  Informal        → Posição: 3 (equilibrado)
Sério   ●●●○○  Divertido       → Posição: 2 (levemente sério)
Técnico ●●●●○  Simples         → Posição: 3 (técnico acessível)
Respeitoso ●●●●●  Irreverente  → Posição: 2 (respeitoso)
Direto  ●●●○○  Narrativo       → Posição: 2 (mais direto)
```

## 2. Persona do Público-Alvo

### Template de Persona

```markdown
## Persona: [Nome]

**Demografia:**
- Idade: [faixa]
- Cargo: [título]
- Empresa: [porte e segmento]
- Localização: [região]

**Psicografia:**
- Valores: [o que importa para ela]
- Desafios: [dores e obstáculos]
- Aspirações: [onde quer chegar]
- Fontes de informação: [onde busca conteúdo]

**Comportamento:**
- Canais preferidos: [LinkedIn, Instagram, email, etc.]
- Horário de consumo: [quando está ativa]
- Formato preferido: [vídeo, texto, áudio]

**Objeções comuns:**
- [Objeção 1 e como contornar]
- [Objeção 2 e como contornar]
- [Objeção 3 e como contornar]
```

## 3. Guia de Linguagem

### Palavras e expressões preferidas

| ✅ Use | ❌ Evite | Motivo |
|--------|----------|--------|
| "Transformar" | "Mudar" | Mais impacto e aspiração |
| "Estratégia" | "Plano" | Transmite profundidade |
| "Resultado" | "Consequência" | Positivo vs neutro |
| "Descubra" | "Veja" | Ativa curiosidade |
| "Comprovado" | "Testado" | Mais confiança |
| "Essencial" | "Importante" | Mais urgência |
| "Construir" | "Fazer" | Transmite processo e solidez |
| "Insight" | "Dica" | Mais valor percebido |

### Expressões proibidas (clichês de marketing)

Nunca use estas expressões sem contexto específico:

- "Solução completa" (vago)
- "Líder de mercado" (sem prova)
- "Inovação disruptiva" (buzzword vazio)
- "One-stop shop" (clichê)
- "Pensar fora da caixa" (gasto)
- "Sinergia" (corporativês)
- "Empoderar" (em excesso)
- "Game changer" (sem evidência)
- "Ecossistema" (quando não se aplica)
- "Next level" (vago)

### Estruturas de frase

| Tipo | Exemplo | Quando usar |
|------|---------|-------------|
| **Frase curta e direta** | "Resultados falam." | Headlines, CTAs, aberturas |
| **Pergunta retórica** | "E se você pudesse prever o churn?" | Engajamento, reflexão |
| **Dados + insight** | "73% dos clientes desistem na 2a tentativa." | Autoridade, prova |
| **Analogia** | "É como ter um CMO 24/7." | Simplificação de conceito |
| **Lista numerada** | "3 passos para..." | Escaneabilidade, promessa |

## 4. Consistência por Canal

### Adaptação de voz (não de personalidade)

A personalidade da marca é fixa. O tom se adapta ao canal:

```
LinkedIn:   Profissional + insights + dados do setor
Instagram:  Visual + inspirador + bastidores + storytelling
Blog:       Educativo + aprofundado + SEO-friendly
Email:      Pessoal + direto + conversacional
Apresentação: Objetivo + visual + orientado a decisão
Proposta:   Formal + personalizado + focado em ROI
```

### Regras universais (todos os canais)

1. **Nunca** falar mal da concorrência diretamente.
2. **Sempre** usar dados para sustentar afirmações.
3. **Sempre** escrever na voz ativa, não passiva.
4. **Nunca** usar jargão excessivo sem explicação.
5. **Sempre** revisar antes de publicar.
6. **Nunca** prometer o que não pode entregar.

---

## 5. Guia de Inclusão e Sensibilidade

### Linguagem inclusiva

- Use linguagem que inclua todos os gêneros.
- Evite estereótipos de gênero, raça, idade ou capacidade.
- Prefira "pessoas" a termos que definam por condição.
- Use "acessível" em vez de "para deficientes".

### Sensibilidade cultural

- Verifique referências culturais antes de usar.
- Evite humor que possa ofender grupos.
- Considere o contexto global ao usar expressões regionais.
- Revise imagens para representatividade diversa.
