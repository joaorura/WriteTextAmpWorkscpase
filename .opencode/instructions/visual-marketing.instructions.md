# Visual Marketing — Diretrizes de Design

> Design visual é comunicação não-verbal. Cada elemento deve ter propósito.
> Consistência visual constrói reconhecimento de marca.

---

## 1. Princípios de Design para Marketing

### Hierarquia Visual

Guie o olhar do espectador na ordem certa:

```
1. Elemento dominante (headline, imagem principal)
2. Elemento secundário (subtítulo, benefício)
3. Elemento terciário (detalhes, CTA)
```

**Regras:**
- Um único ponto focal por peça
- Tamanho indica importância
- Contraste cria atenção
- Espaço em branco é respiro (não desperdício)

### Grid e Alinhamento

- Use grid de 12 colunas para layouts complexos
- Alinhe elementos à esquerda para leitura ocidental
- Margens generosas (mínimo 5% da largura)
- Agrupe elementos relacionados (Lei da Proximidade)

### Contraste e Legibilidade

| Combinação | Legibilidade | Uso |
|------------|--------------|-----|
| Texto escuro + fundo claro | Excelente | Corpo de texto |
| Texto claro + fundo escuro | Bom | Headlines, destaque |
| Texto colorido + fundo colorido | Ruim | Evitar |
| Texto sobre imagem | Variável | Usar overlay/sombra |

**Regra de ouro:** Contraste mínimo de 4.5:1 para texto normal (WCAG AA).

## 3. Tipografia para Marketing

### Hierarquia Tipográfica

```
Headline (H1): 36-72pt — Impacto, atenção
Subheadline (H2): 24-36pt — Complemento, contexto
Body (corpo): 14-18pt — Leitura confortável
Caption (legenda): 10-12pt — Detalhes, créditos
```

### Combinações de Fontes

| Título | Corpo | Estilo |
|--------|-------|--------|
| **Montserrat Bold** | Open Sans Regular | Moderno, acessível |
| **Playfair Display** | Lato Light | Elegante, editorial |
| **Poppins SemiBold** | Roboto Regular | Tech, startup |
| **Bebas Neue** | Source Sans Pro | Impacto, esportivo |
| **Merriweather** | Inter | Clássico, confiável |

### Regras Tipográficas

1. **Máximo 2 famílias** por peça (título + corpo)
2. **Máximo 3 pesos** por família (light, regular, bold)
3. **Tamanho mínimo 12pt** para leitura (10pt para legendas)
4. **Line height 1.4-1.6** para corpo de texto
5. **Evite CAIXA ALTA** em textos longos (dificulta leitura)
6. **Alinhamento à esquerda** para corpo (justificado só em colunas estreitas)

## 4. Cores para Marketing

### Psicologia das Cores

| Cor | Emoção | Uso em Marketing |
|-----|--------|------------------|
| **Vermelho** | Urgência, paixão, energia | CTAs, promoções, liquidações |
| **Azul** | Confiança, segurança, profissional | Corporativo, fintech, saúde |
| **Verde** | Crescimento, natureza, saúde | Sustentabilidade, wellness, finanças |
| **Amarelo** | Otimismo, atenção, cautela | Destaque, alerta, food |
| **Roxo** | Luxo, criatividade, sabedoria | Premium, beleza, educação |
| **Laranja** | Entusiasmo, diversão, acessível | CTAs amigáveis, jovem |
| **Preto** | Sofisticação, poder, elegância | Luxury, moda, premium |
| **Branco** | Pureza, simplicidade, espaço | Minimalismo, respiro |

### Paleta de Marca

Defina para cada projeto:

```markdown
## Paleta de Cores

### Primária
- Cor principal: #XXXXXX (uso: headlines, CTAs)
- Cor secundária: #XXXXXX (uso: elementos de apoio)

### Neutras
- Escuro: #XXXXXX (uso: texto principal)
- Médio: #XXXXXX (uso: texto secundário)
- Claro: #XXXXXX (uso: fundos, bordas)

### Accent
- Destaque: #XXXXXX (uso: links, hover, alertas)
- Sucesso: #XXXXXX
- Erro: #XXXXXX
```

### Regra 60-30-10

- **60%** cor dominante (geralmente neutra — fundos)
- **30%** cor secundária (elementos de apoio)
- **10%** cor de destaque (CTAs, elementos-chave)

## 5. Imagens e Fotografia

### Diretrizes de Imagem

| Tipo | Quando usar | Exemplo |
|------|-------------|---------|
| **Fotografia real** | Autenticidade, conexão humana | Equipe, clientes, bastidores |
| **Ilustração** | Conceitos abstratos, tom leve | Explicações, onboarding |
| **Ícones** | Navegação, features, listas | UI, infográficos |
| **Gráficos/Dados** | Comprovar, informar | Relatórios, cases |
| **Screenshots** | Demonstrar produto | Software, apps |

### Prompts para Geração de Imagem AI (FLUX)

#### Estilo Fotográfico
```
Professional photograph of [subject], [lighting] lighting,
[color palette] color palette, shot on [camera/lens],
editorial style, high resolution
```

#### Estilo Ilustração
```
Modern flat illustration of [subject], [color] color scheme,
clean vector style, minimal details, marketing illustration
```

#### Estilo Banner
```
Marketing banner background, [theme], [colors] gradient,
abstract shapes, professional, 1200x630 aspect ratio
```

### Regras de Imagem

1. **Sempre use alt text** descritivo para acessibilidade
2. **Otimize para web:** JPEG 80-90% qualidade, WebP quando possível
3. **Nomeie arquivos** descritivamente: `banner-lancamento-produto-x.jpg`
4. **Evite stock genérico** — prefira autêntico ou AI customizado
5. **Consistência de estilo** — mantenha o mesmo estilo visual no projeto

## 6. Layout de Apresentações

### Estrutura de Slide

```
┌─────────────────────────────────────────┐
│  [Logo]                    [Nº Slide]   │  ← Header (opcional)
│                                         │
│                                         │
│         CONTEÚDO PRINCIPAL              │  ← Zona nobre (70% do slide)
│                                         │
│                                         │
│  [Fonte/Crédito]                        │  ← Footer (opcional)
└─────────────────────────────────────────┘
```

### Tipos de Slide

| Tipo | Conteúdo | Palavras máx |
|------|----------|--------------|
| **Capa** | Título + subtítulo + logo | 15 |
| **Problema** | Descrição da dor | 30 |
| **Solução** | Como resolvemos | 30 |
| **Dados** | Gráfico + insight | 20 + gráfico |
| **Quote** | Depoimento + foto + nome | 30 |
| **Lista** | 3-5 bullets | 50 |
| **Comparação** | Tabela 2-3 colunas | 40 |
| **CTA** | Ação + contato | 15 |

### Regras de Apresentação

1. **1 ideia por slide** — nunca sobrecarregue
2. **Fonte mínima 24pt** para corpo
3. **Contraste alto** — legível a 3 metros
4. **Animações sutis** — fade in/out, sem exageros
5. **Numere os slides** — facilita referência
6. **Slide de CTA** no final — próximo passo claro

## 7. Checklist de Design

Antes de entregar qualquer peça visual:

- [ ] **Hierarquia** está clara (o que ler primeiro, segundo, terceiro)?
- [ ] **Contraste** é suficiente para legibilidade (4.5:1 mínimo)?
- [ ] **Alinhamento** está consistente (grid, margens)?
- [ ] **Espaço em branco** é adequado (não apertado, não vazio)?
- [ ] **Tipografia** segue a hierarquia (tamanhos, pesos)?
- [ ] **Cores** seguem a paleta do projeto?
- [ ] **Imagens** são relevantes e de alta qualidade?
- [ ] **Dimensões** estão corretas para o canal?
- [ ] **CTA** é visível e claro?
- [ ] **Acessibilidade** foi considerada (alt text, contraste)?
