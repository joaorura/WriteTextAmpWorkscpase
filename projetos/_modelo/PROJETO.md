## Como Trabalhar Neste Projeto

### 1. Leia o Contexto Primeiro
Antes de produzir qualquer conteúdo, leia:
- `briefing.md` — objetivo, escopo, requisitos
- `brand-voice.md` — tom, personalidade, vocabulário
- `personas.md` — quem é o público, o que importa

### 2. Produza Conteúdo
Use os agents e skills disponíveis:
- `@marketing-copywriter` — copy para qualquer canal
- `@marketing-designer` — peças visuais
- `@content-strategist` — planejamento e estratégia
- `@presentation-builder` — apresentações e pitch decks

### 3. Salve no Lugar Certo
- Artigos → `output/artigos/`
- Posts → `output/social-media/`
- Emails → `output/emails/`
- Apresentações → `output/apresentacoes/`
- Imagens → `output/imagens/`
- PDFs/Word → `output/documentos/`

### 4. Gere Documentos Finais
Use os MCPs disponíveis:
- **document-generator** — Word (.docx) + PDF
- **pdf-toolkit** — PDF avançado (merge, encrypt, QR)
- **FLUX** — Imagens AI (free tier disponível)
- **Pictify** — Imagens de templates (free tier disponível)
