# WriteTextAmp Workspace para Codex CLI

> **Carregado pelo Codex CLI ao entrar neste diretório.**

## ⚠️ OBRIGATÓRIO

Leia `.opencode/instructions/using-marketing-workspace.instructions.md` — define o workflow do workspace.

## 🎯 Workflow

1. **Brainstorming** (`brainstorming-content`) se projeto complexo
2. **Planejamento** (`writing-content-plan`)
3. **Execução** com subagents
4. **Revisão** (`reviewing-content`)
5. **Finalização** (`finishing-content-project`)

## 🤖 Subagents

- `content-creator` — Artigos, blogs, whitepapers
- `visual-designer` — Imagens, banners, visuais
- `social-media-manager` — Posts, calendários
- `document-generator` — PDFs, Word, apresentações
- `email-marketing` — Sequências, newsletters

## 🔧 MCPs

Config em `.codex/config.toml`:
- **docling**, **flux**, **pictify**, **changeimageto**, **chompute**, **nano-banana**
- **document-generator**, **pdf-toolkit**
