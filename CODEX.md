## 🎯 Visão Geral

Você está no **WriteTextAmp Workspace**, um ambiente completo de marketing com orquestração inteligente de agents, subagents e skills.

## 🔧 MCPs Configurados

Este workspace tem MCPs pré-configurados em `.codex/config.toml`:

- **Docling** — Processamento de documentos
- **FLUX** — Geração de imagens AI
- **Pictify** — Templates HTML para imagens
- **Open Design** — Prototipagem visual
- **ChangeImageTo** — Edição de imagem
- **document-generator** — Word + PDF
- **pdf-toolkit** — PDF avançado

## 🚀 Workflow

1. **Brainstorming** (`brainstorming-content`) — refina escopo
2. **Planejamento** (`writing-content-plan`) — cria plano
3. **Execução** — delega para subagents
4. **Revisão** (`reviewing-content`) — valida qualidade
5. **Finalização** (`finishing-content-project`) — entrega
